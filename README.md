## 🚀 Overview

A full-stack **Proctored Online Examination System** built using **.NET Core + Angular**.  
It enables secure online tests with real-time monitoring, violation detection, and admin control.

---

## 📌 Features

### 👨‍💼 Admin Panel
- Create, update, delete tests
- Manage questions (MCQs)
- Assign tests to candidates
- Monitor sessions in real-time
- View violations, screenshots & recordings
- Suspend users during exams

### 👨‍🎓 Candidate Panel
- Secure login (JWT)
- View assigned tests
- Attempt exams with timer
- Submit answers
- View results

### 🎥 Proctoring System
- Screenshot capture
- Tab switch detection
- Fullscreen enforcement
- Keyboard & right-click restriction
- Screen + webcam recording
- Auto violation tracking

## ⚙️ Setup Instructions

### Backend

```bash
dotnet restore
dotnet run
```

Runs at: https://localhost:5000

---

### Frontend

```bash
npm install
ng serve
```

Runs at: http://localhost:4200
